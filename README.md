# AIFLoader
[ARM Image Format](http://en.wikipedia.org/wiki/Arm_Image_Format) loader for [Hopper](http://www.hopperapp.com) disassembler

### About
A very quick 'n dirty AIF loader for Hopper.  Supports a minimal subset of AIF â€“ that used by Apple for the Newton. The code and data segments are loaded properly, and the symbols from the AIF data region are parsed.

### Tested
* Newt KNoDebug image
* Senior CirrusNoDebug image
* Egger 'aif ' resource 128
* Hammer 'aif ' resources 128, 130, 131, 1000.
